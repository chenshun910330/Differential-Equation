# 平台競爭動態模型分析：基於 Lotka–Volterra 模型
> **Differential Equations Course Project | Junior Year (3rd Year)**

這是我在大三「微分方程」課程中，與同學合作完成的研究作品。本計畫根據教授要求，從學術期刊中選取與微分方程相關的論文進行深入研究，並分析其數學模型在現實商業環境中的應用與預測能力。

## 📌 專案概述
本研究核心在於探討 **Lotka–Volterra 模型**（原用於描述生物種群競爭）如何被引入經濟學領域，用以模擬「雙邊平台」（如美食外送、網約車）之間的動態競爭、市場壟斷與穩定共存條件。

### 核心研究亮點 (Highlights)
* **關鍵增長率**：用戶內生增長率對「壟斷區域」的影響遠大於其他因素。
* **優勢佔領**：具有較高增長率的平台在競爭中極易達成市場壟斷，取得短期流量優勢是最佳策略。
* **穩定共存**：在滿足特定跨網路外部性條件下，競爭平台可維持穩定的市場結構。
* **進入障礙**：即使新平台具備強大的網路外部性，若無法突破既有雙頭壟斷（Duopoly）的壓力門檻，仍難以立足市場。

## 🧮 數學模型 (Mathematical Model)
本專案分析的模型基於非線性微分方程組，描述兩平台用戶規模 $x_1, x_2$ 隨時間的演化：

$$
\begin{cases} 
\frac{dx_1}{dt} = r_1 x_1 \left(1 - \frac{x_1}{K_1} - \alpha_{12} \frac{x_2}{K_2}\right) \\
\frac{dx_2}{dt} = r_2 x_2 \left(1 - \frac{x_2}{K_2} - \alpha_{21} \frac{x_1}{K_1}\right) 
\end{cases}
$$

* **$r_i$**: 平台 $i$ 的內生增長率 (Intrinsic Growth Rate)。
* **$K_i$**: 平台市場飽和容量。
* **$\alpha_{ij}$**: 跨網路外部性所產生的抑制或競爭係數。



## 📊 實證分析：美團 vs. 餓了麼
研究最後針對中國外送市場兩大巨頭 —— **美團 (Meituan)** 與 **餓了麼 (Ele.me)** 進行實證分析：
1.  **理論驗證**：利用歷史數據回歸模型參數，驗證微分方程組的準確性。
2.  **趨勢預測**：根據當前增長率與競爭壓力，預測市場將趨於「動態共存」或「單一壟斷」。
